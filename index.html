LibN3L

Recently I developed a libary for neural network in NLP myself to faciliate my research, which is an advanced version of my LREC paper. It is now in experimental phase. The main framework is stealed from cnn by Chris Dyer. The libary has the following attributes:
  • Graph-based framework that all features are organized by a graph.
  • Support discrete and neural features, treating them equally.
  • Forward is used to build a graph.
  • Output nodes and their losses are defined according to requirements manually.
  • Backward is excuted automatically, and will add multithread implementation for it.
  • Sample codes: CRF and Transition-based framework for Chinese word segmentation.
