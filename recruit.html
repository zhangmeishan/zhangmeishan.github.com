招生说明

  • 每年招收硕士生2~3名（推免为主），欢迎提前联系，最终录取决定权在学院。
    1. 硕士生主要由博士生协助指导，师兄师姐手把手教学，导师定期参与讨论；
    2. 硕士生毕业要求发表一篇 CCF C 以上一作论文，同时帮助课题组完成一定的其它任务；
    3. 研究-非研究比例大致介于64 到73开之间；
    4. 满足毕业条件一般可以通过课题组内部推荐实习；
    5. 组里氛围相对融洽，多位天大学生跟过来一起读博（读博名额少，很多学生有一年GAP）。
  • 与苏州大学李正华老师联合指导的硕士生
    1. 自我感觉哈工深比较难的可以考虑这里，发邮件请注明；
    2. 必须两个月实践考察期，非竞争型，只要满足我和李正华老师的要求即可；
    3. 本质上是李正华老师学生，属于共同指导，做我们共同感兴趣的事情。
  • 实习生多名，与组内博士生协作完成研究课题或实践项目。
    1. 主要面向哈工大深圳本科生；
    2. 其它学校能保证时间远程也可以，主要由博士生项目驱动，初期以实践项目开发为主（特别优秀的大佬请慎重考虑）；
    3. 要求具有一定编程基础，课题内容对应组内主要研究方向，可逐步参与研究项目，有一定补助；
    4. 同时欢迎本校学生依托开展本科生创新项目。
    若有意向，请电子邮件发送自我介绍和简历到 thereiswater@googlegroups.com 。
    标题格式：【推免/实习】姓名-学校-感兴趣的方向。采取batch的回复策略，定期批量处理申请。

    课题方向：围绕RAG和LLM开展研究。
    1. 句法语义分析：成分句法、依存句法、语义分析，及结合 LLM 和多模态。相关研究曾以第一或通讯作者发表ACL、AI、CL等顶级会议和期刊论文多篇。
    2. 代码智能：代码大模型、代码翻译、代码&文档问答等。是课题组重点发展的新方向。
    3. 表示学习和信息检索：文本和多模态 embedding、retrieval，有状态的LLM智能系统。研发了国际领先的text embedding模型，huggingface月下载量达150万次。
    4. 情感分析：LLM结合多模态、多智能体以及检索增强生成（RAG）。相关研究曾发表AAAI、WWW、EMNLP等顶级会议和期刊论文多篇。
    5. 多模态生成与理解：多模态内容的联合生成、结构化知识驱动的多模态内容理解。相关研究曾以通讯作者发表TPAMI、ICML、ACL、MM等顶级会议和期刊论文多篇。
